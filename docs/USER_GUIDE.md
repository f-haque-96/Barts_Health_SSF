## Overview

This guide explains how to submit a new supplier setup request through the NHS Supplier Setup Form. The process ensures all suppliers are properly vetted and registered before being added to the system.

**Total Time Required:** 20-30 minutes
**Prerequisites:** NHS email address, supplier information, required documents - Bank details of supplier on their official letterhead.

**Security Note:** All information you submit is encrypted, validated, and securely stored. Your submission is protected by multiple layers of security added in February 2026.

## Before You Start

### Information You'll Need

1. **Your Information:**
   - Full name
   - Job title
   - Department
   - NHS email address
   - Phone number

2. **Supplier Information:**
   - Company name
   - Company Registration Number (CRN) if applicable
   - Registered address
   - Contact person details
   - Type of supplier (Limited Company, Sole Trader, Charity, etc.)

3. **Service Details:**
   - What goods/services they will provide

- Procurement approval certificate (You can obtain this by completing the clinical/ non clinical questionnaire on the form Q2.7)

**Format:** PDF only
**Max File Size:** 3MB per file

## Step-by-Step Process

### Section 1: Your Details

1. Navigate to the NHS Supplier Setup Form homepage
2. Click "Start New Request"
3. Fill in your personal information:
   - First Name
   - Last Name
   - Job Title
   - Department (select from dropdown)
   - NHS Email
   - Phone Number
4. Click "Next" to proceed

**Note:** All fields are mandatory. Use your official NHS email address.

---

### Section 2: Pre-Screening Questions

This section determines if procurement approval is required.

1. **Service Category:** Select the type of service/goods
   - Facilities & Estates
   - IT & Technology
   - Clinical Services
   - Professional Services
   - Goods & Supplies

2. **Procurement Engagement:**
   - Have you already engaged with procurement? (Yes/No)
   - If YES: Upload procurement approval document

3. **Letterhead:**
   - Do you have letterhead with bank details? (Yes/No)
   - If YES: Upload letterhead (REQUIRED)

4. **Sole Trader Status:**
   - Is this a sole trader or contractor? (Yes/No)
   - If YES: Upload CEST form

5. **Usage Frequency:**
   - One-time purchase
   - Occasional use
   - Regular/ongoing

6. **Supplier Connection:**
   - Do you have any connection to this supplier? (Yes/No)
   - If YES: Provide details (conflict of interest declaration)

**Important:** If you have NOT engaged with procurement and the purchase value is over ¬£5,000, you MUST submit a questionnaire to the Pre-Buy Panel (PBP) for approval before continuing.

#### Pre-Buy Panel Questionnaire (If Required)

If prompted, you'll need to answer these questions:

1. **Purchase Justification:** Why is this purchase necessary?
2. **Alternatives Considered:** What other options did you explore?
3. **Budget Confirmation:** Is budget available?
4. **Estimated Value:** What is the expected cost?
5. **Urgency Reason:** Why is this urgent (if applicable)?

After submission:
- PBP will review within 3-5 working days
- You'll receive an email notification with their decision
- If approved: upload to the form and proceed to Section 3
- If rejected: Review feedback and resubmit
- If more info needed: Respond and await re-review

---

### Section 3: Supplier Classification

1. **Companies House Registration:**
   - Is the supplier registered with Companies House? (Yes/No)

2. **Supplier Type:** Select from:
   - Limited Company
   - Sole Trader
   - Partnership
   - Charity
   - Public Sector Organisation

3. **Company Registration Number (CRN):**
   - Enter CRN if Limited Company registered with Companies House
   - System will auto-verify and populate company details
   - If verification fails, check the number and try again

4. **Additional Fields (based on type):**
   - **Charity:** Charity number required
   - **Sole Trader:** Select ID type (Passport or Driving Licence), upload photo ID
   - **Public Sector:** Select organisation type

5. **Supplier Metrics:**
   - Annual value of contract
   - Number of employees

Click "Next" when complete.

---

### Section 4: Supplier Details

Enter complete supplier information:

1. **Company Name**
2. **Trading Name** (if different)
3. **Registered Address**
4. **City**
5. **Postcode**
6. **Contact Person Name**
7. **Contact Email**
8. **Contact Phone**
9. **Website** (optional)

**Tip:** If you verified the CRN in Section 3, some fields may auto-populate.

---

### Section 5: Service Description

1. **Service Type:** Select all that apply (multi-select):
   - Goods
   - Services
   - Construction
   - Consultancy
   - Temporary Staff

2. **Service Description:**
   - Provide detailed description of what the supplier will provide
   - Include scope, deliverables, and frequency
   - Minimum 50 characters

---

### Section 6: Financial Information

#### Bank Details

1. **Overseas Supplier:** Is this a non-UK supplier? (Yes/No)
   - If YES: Provide IBAN, SWIFT code, Bank routing number
   - If NO: Letterhead with UK bank details required

#### Accounts Payable Contact

2. **Same as Registered Address?** (Yes/No)
   - If NO: Provide separate accounts address and contact details

#### Additional Financial Information

3. **GHX/DUNS Number:**
   - Do you know the GHX or DUNS number? (Yes/No)
   - If YES: Enter number

4. **CIS Registration:**
   - Is supplier registered for Construction Industry Scheme? (Yes/No)
   - If YES: Provide UTR number

5. **Public Liability Insurance:**
   - Does supplier have public liability insurance? (Yes/No)
   - If YES: Coverage amount and expiry date

6. **VAT Registration:**
   - Is supplier VAT registered? (Yes/No)
   - If YES: Provide VAT number

---

### Section 7: Review & Submit

1. **Review all information:**
   - Check each section for accuracy
   - Edit any section by clicking "Edit Section X"
   - Ensure all mandatory fields are complete
   - Verify all documents are uploaded

2. **Missing Information Alert:**
   - Red banner will appear if any required fields are missing
   - Click "Show Details" to see specific missing items
   - Complete missing information before proceeding

3. **Data Accuracy Confirmation:**
   - Read the declaration
   - Check "I confirm that all information provided is accurate"

4. **Submit:**
   - Click "Submit Supplier Request"
   - You'll receive a submission ID (format: SUP-2026-XXXXX)
   - Save this ID for tracking

5. **PDF Download:**
   - Click "Download PDF" to save a copy of your submission
   - Keep this for your records

## Common Issues & Solutions

### Issue: CRN Verification Failed

**Cause:** Incorrect CRN or company not found
**Solution:**
1. Verify CRN on Companies House website
2. Check for leading zeros or spaces
3. Ensure company is active (not dissolved/inactive/liquidated)
4. If still failing, continue manually.

### Issue: File Upload Failed

**Cause:** File too large or wrong format
**Solution:**
1. Check file is PDF format
2. Ensure file size is under 3MB
3. Compress PDF if needed (use Adobe Acrobat or online tool)
4. Rename file (remove special characters)

### Issue: Cannot Submit - Missing Fields

**Cause:** Required information not completed
**Solution:**
1. Look for red "Missing Information" banner
2. Click "Show Details" to see list
3. Navigate to relevant section
4. Complete all mandatory fields
5. Return to Section 7 and try again

### Issue: Pre-Buy Panel Rejected Request

**Cause:** Insufficient justification or compliance issues
**Solution:**
1. Read rejection comments carefully
2. Address all concerns raised
3. Gather additional supporting information
4. Submit new request with updated justification

### Issue: Form Data Lost After Browser Refresh

**Cause:** Session expired or browser cache cleared
**Solution:**
- Form auto-saves to local storage
- Data should persist across refreshes
- If lost, re-enter information
- Upload documents again
- Consider completing form in one session

## FAQs

### General Questions

**Q: How long does the approval process take?**
A: Typically 5-10 working days for standard suppliers. Sole traders requiring OPW/IR35 assessment may take 10-15 days.

**Q: Can I save my progress and return later?**
A: Yes, the form auto-saves to your browser. Return to the same browser on the same computer to continue.

**Q: Can I submit multiple suppliers at once?**
A: No, each supplier requires a separate submission. Complete one form per supplier.

**Q: What if I don't have all the information?**
A: Gather all required information before starting. Contact the supplier directly for missing details.

**Q: Can I update information after submission?**
A: No, you cannot edit after submission. If critical information is wrong, contact supplier-admin@nhs.net with your submission ID.

### Document Questions

**Q: What if the supplier doesn't have letterhead?**
A: Request they provide a signed letter on company stationery with bank details. If truly no letterhead exists, contact procurement team for guidance.

**Q: Can I submit documents in Word or Excel format?**
A: No, only PDF format is accepted. Convert documents to PDF before uploading.

**Q: The letterhead doesn't show bank details. What do I do?**
A: Request a separate bank letter from the supplier or ask them to add bank details to their letterhead.

### Procurement Questions

**Q: Do I need procurement approval?**
A: If you've already engaged with the procurement team, upload their approval. If not and value is over ¬£5,000, you'll need Pre-Buy Panel approval (handled in Section 2).

**Q: What if procurement rejected my request?**
A: Address their concerns before submitting the supplier form. You need procurement/PBP approval to proceed.

### Technical Questions

**Q: Which browsers are supported?**
A: Chrome, Edge, Firefox, Safari (latest versions). Internet Explorer is NOT supported.

**Q: Why can't I see my uploaded files?**
A: Files are stored in browser memory. If you clear cache or switch browsers, you'll need to re-upload.

**Q: The form is not loading properly. What should I do?**
A: Clear browser cache, try incognito mode, or use a different browser. Contact IT support if issues persist.

### Process Questions

**Q: What is OPW/IR35 assessment?**
A: For sole traders and contractors, the Off-Payroll Working panel assesses whether they should be treated as employees for tax purposes.

**Q: Who reviews my submission?**
A: Depends on supplier type:
- All submissions: Procurement Team
- Sole traders: OPW/IR35 Panel
- All suppliers: AP Control (final setup)

**Q: What happens if my request is rejected?**
A: You'll receive an email with rejection reasons. Address the issues and submit a new request.

**Q: Can I expedite an urgent request?**
A: Indicate urgency in the questionnaire. However, all requests follow the standard review process. Plan ahead for new suppliers.

### Security Questions (New - Feb 2026)

**Q: Is my data secure?**
A: Yes! The system now includes multiple security layers added in February 2026:
- All data is encrypted
- All actions are logged (audit trail)
- Sensitive documents (passports, driving licences) are stored separately and auto-deleted after 30 days
- Bank details are protected with enhanced encryption

**Q: Will I be notified if something goes wrong with security?**
A: Yes. If any security validation fails (e.g., suspicious file upload, duplicate supplier detected), you'll be notified immediately via email.

**Q: What happens to sensitive documents like my passport copy?**
A: For sole traders:
- Passport/driving licence copies are stored in a secure, restricted SharePoint library
- Only AP Control and Admin can access these documents
- They are automatically deleted 30 days after your supplier is set up
- Every access is logged for GDPR compliance

**Q: How do I know my submission wasn't tampered with?**
A: The system now creates an audit trail showing:
- When you submitted
- What you submitted
- Who reviewed it
- What decisions were made
- When changes occurred
This cannot be altered or deleted, ensuring full transparency.
